# 🚀 微信小程序发布与上传指南

按照以下步骤，你可以将舒尔特方格游戏发布到微信小程序平台上。

## 第一步：账号注册
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)。
2. 点击右上角“立即注册”。
3. 选择“小程序”。
4. 按引导完成注册（个人类型是免费的）。

## 第二步：获取 AppID
1. 登录微信公众平台。
2. 在左侧菜单找到 **设置** -> **基本设置**。
3. 复制你的 **AppID**。

## 第三步：配置项目
1. 在 VS Code 中打开 `schulte-game-wechat/project.config.json`。
2. 将 `"appid": "touristappid"` 中的 `touristappid` 替换为你真实的 AppID。

## 第四步：上传预览
1. 在 **微信开发者工具** 中打开此项目。
2. 点击顶部工具栏右侧的 **预览**，用手机微信扫描二位码即可体验。
3. 点击 **上传** 按钮，填写版本号（如 1.0.0）并确认。

## 第五步：提交审核
1. 回到微信公众平台网页版。
2. 菜单选取 **版本管理**。
3. 在“开发版本”中找到刚才上传的项目，点击“提交审核”。
4. 填写相关表单，等待微信团队审核（通常 1-3 天）。
5. 审核通过后，在版本管理中点击“发布”。

## 💡 特别提醒
- **免费说明**：只要不进行企业认证，个人小程序的注册和发布都是 0 成本的。
- **审核注意**：舒尔特方格属于“工具-教育”类目或“游戏”，审核非常容易通过。
